    Model.find

    Model.find(conditions, projection, options, callback);

    Params

    • conditions 筛选条件
    • projection 换回的数据只包含这些字段
    • options 游标操作
    • callback 回调函数，返回结果后调用

    Model.findById

    通过唯一 ID 查询文档，findById(id) 和 findOne({ _id: id }) 几乎相同。

    Model.findById(id, projection, options, callback);
